# Task Flow

A modern and responsive Todo App built with **React**, **JavaScript**, **HTML**, and **Tailwind CSS**.


## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks by status
- View total active tasks
- Clean responsive UI


## Tech Stack

- React
- JavaScript
- HTML5
- Tailwind CSS

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/ahmud-z/task-flow.git
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```
